# MCP Filesystem Server

**Repository**: [modelcontextprotocol/servers/src/filesystem](https://github.com/modelcontextprotocol/servers/tree/main/src/filesystem)

## Overview
The Filesystem server enables secure file operations with configurable access controls. It allows the AI to read, write, and manage files within specified directories.

## Features
-   Read files and directories.
-   Write and edit files.
-   List directory contents.
-   Check file existence and metadata.

## Usage
```bash
npx -y @modelcontextprotocol/server-filesystem /path/to/allowed/directory
```
