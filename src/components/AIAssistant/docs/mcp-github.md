# MCP GitHub Server

**Repository**: [modelcontextprotocol/servers/src/github](https://github.com/modelcontextprotocol/servers/tree/main/src/github)

## Overview
The GitHub server enables interaction with the GitHub API. It allows the AI to manage issues, pull requests, and repository data.

## Features
-   Search repositories.
-   Read and create issues.
-   Manage pull requests.
-   Inspect file content on GitHub.

## Usage
```bash
npx -y @modelcontextprotocol/server-github
```
