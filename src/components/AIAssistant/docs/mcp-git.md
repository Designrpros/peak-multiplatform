# MCP Git Server

**Repository**: [modelcontextprotocol/servers/src/git](https://github.com/modelcontextprotocol/servers/tree/main/src/git)

## Overview
The Git server provides tools to read, search, and manipulate Git repositories. It enables the AI to understand the history and state of your codebase.

## Features
-   Read git logs and history.
-   Diff commits and branches.
-   Check status of the working directory.
-   (Optional) Commit and push changes.

## Usage
```bash
npx -y @modelcontextprotocol/server-git
```
