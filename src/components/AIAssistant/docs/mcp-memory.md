# MCP Memory Server

**Repository**: [modelcontextprotocol/servers/src/memory](https://github.com/modelcontextprotocol/servers/tree/main/src/memory)

## Overview
The Memory server implements a knowledge graph-based persistent memory system. It allows the AI to store and retrieve structured information across sessions.

## Features
-   Store entities and relationships.
-   Query the knowledge graph.
-   Persist context beyond a single conversation.

## Usage
```bash
npx -y @modelcontextprotocol/server-memory
```
